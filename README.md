# meal-website-
The MealDB App is a dynamic, responsive single-page web application designed to help users discover culinary ideas and detailed recipes from around the world. Powered by the external MealDB API, the application allows users to seamlessly search for specific dishes, browse by global cuisine, or explore meals based on specific ingredients.The project focuses on providing a smooth, interactive user experience through asynchronous data fetching, intuitive filtering, and a mobile-friendly design.

Key Features
Smart Search: Users can search for specific meals by name, with a dedicated results interface that updates dynamically.

Discovery & Exploration: The app automatically fetches and displays a grid of random meals upon loading to inspire users immediately.

Ingredient Filtering: Features interactive sections for both popular and randomly generated ingredients. Clicking an ingredient filters the database to show only meals containing that specific item.

Global Cuisine Browsing: Users can explore recipes by country of origin using interactive national flags.

Alphabetical Index: A built-in A-Z glossary allows users to browse the database by the first letter of the meal.

Detailed Recipe Modal: Clicking on any meal card opens a pop-up modal displaying a high-quality image, a formatted list of exact ingredients and measurements, and step-by-step cooking instructions.

Responsive UI/UX: The interface is fully optimized for all devices, utilizing CSS Grid/Flexbox and media queries to ensure a seamless experience on both desktop and mobile screens. It also includes visual feedback, such as a custom loading spinner during API calls.

Technical Stack
Frontend: HTML5, Vanilla JavaScript (ES6+), and CSS3.

Data Integration: RESTful API integration using JavaScript fetch and asynchronous programming (async/await, Promises).

Architecture: Single Page Application (SPA) structure with dynamic DOM manipulation.
